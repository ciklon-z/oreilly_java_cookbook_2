JabaDot: R.I.P.

The JabaDot program has been retired, and is no longer included in this collection. The code is no longer available; it was not well structured, and had become unmaintainable. For a better replacement, check out the Java Lobby's JLCP, at http://www.javalobby.org/.

So then, what's this? Some of these files were used as examples in the Java Cookbook, and I decided to keep them. What's here is just enough to make the files that are used in the book compile!
